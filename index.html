Introduction

JavaScript is a versatile, lightweight, cross-platform scripting language commonly used for enhancing web pages with interactive features.

It integrates with various host environments like browsers, offering extended functionalities.

  • Client-side JavaScript extends the core language by controlling the DOM and responding to user events.
  • Server-side JavaScript adds capabilities for interacting with databases, managing files, and handling server requests.

JavaScript's popularity has grown because of its simplicity and ability to create dynamic, engaging web applications.

console.log('JavaScript is fun!');

Hello World

Here's a simple JavaScript example to get started:

function greet(name) {
    alert(`Hello, ${name}!`);
}
greet('World');

Open your browser console, paste the code, and see it in action!

  • JavaScript is interpreted by browsers.
  • Functions are blocks of code designed to perform a task.

Variables

Variables are used to store data values in JavaScript.

There are different ways to declare variables: using `var`, `let`, and `const`.

  • `var`: Function-scoped, can be re-declared.
  • `let`: Block-scoped, cannot be re-declared.
  • `const`: Block-scoped, cannot be re-assigned.

It's recommended to use `let` and `const` in modern JavaScript for better scoping.

let x = 10;
Functions

Functions are fundamental blocks of JavaScript code that are used to perform tasks or calculations.

You define a function using the `function` keyword followed by a name, parameters, and a block of code.

  • Functions can take inputs (parameters) and return outputs.
  • They help organize code into reusable blocks.

Here's a simple example of a function:

function sum(a, b) { return a + b; }

Call the function like this: `sum(2, 3);`
